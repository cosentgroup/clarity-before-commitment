# Clarity Before Commitment

> Understand the business before you choose it.

A decision clarity platform that helps people make informed decisions about entering demanding businesses — before committing time, capital, or identity.

## Tech Stack

- React 18
- React Router v6
- Create React App
- Deployed on Vercel

## Local Development

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
```

## Deploy

This project is configured for Vercel. Push to GitHub and connect the repository in your Vercel dashboard. Vercel will auto-detect Create React App and deploy automatically on every push to `main`.

The `vercel.json` file handles SPA routing so all page routes work correctly.
