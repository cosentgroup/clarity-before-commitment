# Clarity Before Commitment

> Understand the business before you choose it.

A decision clarity platform helping people make informed decisions about entering demanding businesses — before committing time, capital, or identity.

## Stack
- React 18 + React Router v6
- Create React App
- Deployed on Vercel

## Local Development
```bash
npm install
npm start
```

## Deploy
Push to GitHub. Connect repo in Vercel dashboard. Auto-deploys on every push to `main`.
